# Template: TypeScript

![Pull Request Workflow Status](https://github.com/mikelittman/template-ts/actions/workflows/pull-request.yml/badge.svg)


This is a template for a TypeScript project.

## Usage

### Install

```sh
npm i
```

### Test

```sh
npm t
```

### Type check

```sh
npm run tsc
```

### Build

```sh
npm run build
```
